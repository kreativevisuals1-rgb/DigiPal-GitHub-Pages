# DigiPal GitHub Pages Deployment\n\nThis folder contains the static index.html file for the DigiPal landing page.\n\n## Deploy to GitHub Pages\n1. Create a new GitHub repository.\n2. Push this folder as the repository root.\n3. In repository Settings ? Pages, select the main branch and root folder.\n4. Visit the published URL under GitHub Pages settings.\n\n## Notes\n- If you have a custom domain, add a CNAME file with the domain name.\n- Update SUPABASE_URL and SUPABASE_ANON_KEY in the script before publishing if you want auth to work live.
