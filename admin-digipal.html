📋 How It Works (Feature Rules)

  • Every user gets 5 unique, AI-generated posts every day — no two users see the same content
  • Users read each post for 2 minutes (timer enforced)
  • They write a 500+ character comment after reading
  • AI verifies the comment is genuine before awarding credits
  • Each post rewards a flat 100 credits (= ₦100 equivalent)
  • Max 500 credits per user per day (5 posts × 100 credits = ₦500 equivalent)
  • Credits redeemable for airtime, data, and cable TV only — no cash withdrawal
  • Monthly top commenter wins ₦5,000 paid to their DigiPal wallet
